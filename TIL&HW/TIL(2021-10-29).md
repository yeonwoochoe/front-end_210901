#### ✔**VS CODE 단축키**

```
- shift+alt + 방향키 : 복사

- ctrl+[ ] : 같이 한 번에 잡아 들여쓰기  

- ctrl +  D :  같이 선택

- increment 플러그인 설치  (숫자 자동 변경)

- ctrl+shift+엔터  : 위 방향으로 벌려주기

- alt+방향키 :  위아래 체인지 가능

- ctrl+shift+L : 같은 단어선택 가능

- ctrl+alt+i : 숫자 자동정렬

- alt : 다중선택가능

- ctrl+enter : 그 자리에서 내려가기

- ctrl+ali+방향키 : 문서 나누기
```
